# Quickstart: Physical AI & Humanoid Robotics Course

This guide provides the steps to get the course materials up and running locally.

## Prerequisites

- Node.js (v18 or higher)
- Yarn

## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/physical-ai-humanoid-robotics.git
    cd physical-ai-humanoid-robotics
    ```

2.  **Install dependencies**:
    ```bash
    yarn install
    ```

## Running the Course Locally

1.  **Start the Docusaurus development server**:
    ```bash
    yarn start
    ```

2.  **Open the course in your browser**:
    Navigate to `http://localhost:3000` to view the course materials.
